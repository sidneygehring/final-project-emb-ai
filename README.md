# Emotion Detection Web Application

A web-based application that detects the emotional content of text statements. Users can input text and receive emotion scores for **anger, disgust, fear, joy, and sadness**, as well as the **dominant emotion**.

---

## Features

- Analyze text for multiple emotions.
- Web interface built with Flask.
- Error handling for blank or invalid input.
- Modular design for easy reuse and testing.
